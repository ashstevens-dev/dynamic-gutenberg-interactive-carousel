Interactive carousel of latest posts or products filtered by category.

== Description ==

Dynamic block featuring an interactive carousel of latest posts or products filtered by category. Includes three unique carousel templates to choose from.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/interactive-carousel` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
